La bomba cancella tutti i file del PC vittima.
E' il file zippato "deleteALL", per farla funzionare devi prima eseguire la procedura di "unZip" cliccando con il tasto destro del mouse sul file "deleteALL" --> dal menù a tendina che si apre clicca su estrai (estrai qui o estrai tutto, ecc...).
Se il pc non ha software in grado di estrarre i file zippati (tipo: winZip, 7zip, altri...) --> allora ti suggerisco di installare 7zip --> è gratuito e funziona bene. Buona fortuna.
WARNING: non fatelo girare sul vostro pc e funziona solo su sistemi operativi windows
